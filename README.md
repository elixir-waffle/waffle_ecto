[Documentation](https://hexdocs.pm/waffle_ecto)

## Installation

Add the latest stable release to your `mix.exs` file:

```elixir
defp deps do
  [
    {:waffle_ecto, "~> 0.0.9"}
  ]
end
```

Then run `mix deps.get` in your shell to fetch the dependencies.
